# LocalDocSearch - サービス概要

## 基本情報

| 項目 | 値 |
|------|-----|
| 表示名 | LocalDocSearch |
| GitHubプロジェクト名 | local-doc-search |
| 略称 | LDS |

## エレベーターピッチ

Mac上の画像・PDF・動画・音声を自然言語で検索できるローカルAI検索システム

## 解決する課題

- 「あの資料どこだっけ」問題
- 大量のローカルファイルから目的の情報を見つけられない
- 動画や音声の中身を検索できない
- クラウドサービスにデータを預けるプライバシー懸念

## 提供する価値

1. **自然言語検索**: ファイル名を覚えていなくても内容で検索可能
2. **マルチメディア対応**: 画像はAIが説明文生成、動画・音声は自動文字起こし
3. **タイムスタンプ検索**: 動画・音声内の特定の発言箇所に直接ジャンプ
4. **完全ローカル処理**: データは一切外部に送信しない

## 競合との差別化

| 特徴 | LocalDocSearch | Spotlight | Alfred |
|------|----------------|-----------|--------|
| 自然言語検索 | Yes | No | No |
| 画像内容検索 | Yes (VLM) | Limited | No |
| 動画文字起こし | Yes | No | No |
| ベクトル検索 | Yes | No | No |
| 完全ローカル | Yes | Yes | Yes |

## コンセプト

**完全ローカル処理によるプライバシー保護と高速検索**

すべての処理をMac Studio (M3 Ultra) のローカル環境で完結させ、外部サービスへのデータ送信を一切行わない。
