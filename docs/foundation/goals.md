# プロジェクトゴール

## プロジェクトスコープ

| 項目 | 値 |
|------|-----|
| フェーズ | MVP → フル機能 |
| 利用形態 | 個人利用 |

## 成功指標（KPI）

| 指標 | 目標値 |
|------|--------|
| 検索レイテンシ | < 100ms（100万ドキュメント） |
| インデックス速度（テキスト） | 10ファイル/秒 |
| インデックス速度（画像） | 2-3秒/画像 |
| インデックス速度（動画1時間） | ~5分 |
| メモリ使用量（常駐時） | < 8GB |

## 制約条件

### ハードウェア制約

| 項目 | 制約 |
|------|------|
| 対象マシン | Mac Studio (M3 Ultra) 限定 |
| 最小メモリ | 192GB統合メモリ |
| 最小ストレージ | 十分な空き容量 |

### 技術制約

| 項目 | 制約 |
|------|------|
| 外部通信 | 禁止（完全ローカル処理） |
| LLM | Ollama経由のローカルモデルのみ |

### 運用制約

| 項目 | 制約 |
|------|------|
| 最大ファイルサイズ | 500MB |
| 最大動画長 | 180分 |
| 最大画像サイズ | 4096px |

## 多言語対応

| 項目 | 値 |
|------|-----|
| 対象言語 | 日本語（メイン）、英語 |
| 優先順位 | 日本語のみで開始 |
| ローカライズ範囲 | UI（将来） |
| 翻訳方式 | N/A（個人利用） |

## 非機能要件

### パフォーマンス

| 項目 | 目標 |
|------|------|
| ストレージ効率 | 元ファイルの10-20% |
| 同時処理数 | 最大4ワーカー |

### 信頼性

- **データ整合性**: LanceDBのトランザクション保証
- **クラッシュリカバリ**: WAL（Write-Ahead Logging）
- **バックアップ**: 日次でインデックスDBをバックアップ

### セキュリティ

- **完全ローカル処理**: データは一切外部に送信しない
- **API認証**: ローカルホストのみアクセス可能
